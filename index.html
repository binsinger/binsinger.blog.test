总结一下如何处理搭建一个静态blog

github pages

  • 在github上新建一个 username.github.io 仓库
  • clone到本地,
  • 新建一个 index.html 文件,添加一些内容和样式
  • do “add” and “commit”
  • do push
  • 这样就可以在此域名上访问了(for example:binsinger.github.io)

使用 hexo 结合github pages搭建一个blog

use npm insatll 运行本地服务端

配置

  • 在_config.yml 文件中设置配置,找到deploy 配置如下:
    deploy:
    type: git
    repo: https://binsinger:12345678@github.com/binsinger/binsinger.github.io.git
    //https://username:password@repositoryurl
    //assure this repository is clean

安装 hexo-deployer-git

创建一篇新blog

  • 不结束服务端cmd窗口,在blog目录下运行新的cmd命令窗口

生成后自动部署网站

hexo deploy
//ususally use
hexo generate --deploy
hexo deploy --generate
